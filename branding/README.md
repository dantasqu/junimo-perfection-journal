# Branding

This folder tracks the visual direction for Junimo Perfection Journal.

## Layout

- `current/`: the latest repo-ready artwork
- `site/`: files actively used by the live site UI
- `social/`: GitHub/social preview exports
- `concepts/`: alternate drafts and experiments
- `references/`: source/reference images
- `ICON_BRIEF.md`: art direction notes

## Current Direction

- Open Stardew-style book or journal
- A real Junimo inside the open pages
- Perfection glow in soft honey gold / magical amber
- Cozy, readable app-icon silhouette

## Current Files

- `current/journal-art-transparent.png`: latest transparent artwork from Downloads

- `site/hero-junimo-sprite.png`: current hero artwork used on the live site
